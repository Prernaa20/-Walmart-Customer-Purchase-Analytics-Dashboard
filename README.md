# -Walmart-Customer-Purchase-Analytics-Dashboard
 Just completed my Power BI  assignment and I’m really proud of how it turned out!
Walmart Customer Purchase Analytics Dashboard (2024–2025) – Fully interactive, single-page, with built-in 30-day sales forecasting
Built using real 50,000-transaction dataset, here’s what the dashboard reveals:
$12.77M total revenue
50.48% customers are repeat buyers (strong loyalty!)
Electronics + Beauty = 51% of total sales
36–45 age group spends the most
Discounts increase average order value by ~12%
30-day forecast predicts stable $1.05M revenue in Feb 2026
Features:
6 synchronized slicers
Time-series forecasting (Power BI Analytics pane)
Top 10 products, customer segmentation, payment trends
Clean Walmart-themed design
Learned so much about DAX, Power Query transformations, visual storytelling, and predictive analytics in Power BI
Thank you to my professor and classmates for the motivation!
